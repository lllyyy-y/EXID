# EXID: Explainable Diet–Exercise–Weight Change Model

This repository contains the code and manuscript for the paper submitted to KIPS 2025.

## Repository Structure
- `paper/` : Final manuscript (HWP)
- `code/` : Modeling notebooks (CatBoost, AutoML)
- `data/` : Not publicly available
- `results/` : Figures used in the paper (optional)

## Models
- CatBoost (main model)
- AutoML (benchmark model)

## Data Availability
Due to privacy and licensing issues, the raw dataset is not publicly available.

## Note
This repository is intended for academic and reproducibility purposes.
